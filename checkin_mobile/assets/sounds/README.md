# Sound Assets

Place the following sound files in this directory:

- `success.mp3` - Short confirmation sound for successful check-in
- `error.mp3` - Short error sound for failed check-in

Recommended sources for royalty-free sounds:
- https://freesound.org
- https://pixabay.com/sound-effects
- https://mixkit.co/free-sound-effects

Keep files small (<100KB) for fast loading.
